# ClickBox

내역 추가 모달에서 사용되어지는 클릭 박스입니다.

## Props

```ts
interface ClickBoxProps {
  /**
   * 클릭 박스에서 다루는 데이터 텍스트
   */
  placeholder: string;
  /**
   * 오른쪽 아이콘을 표시할지 여부
   */
  isRightIcon?: boolean;
  /**
   * 클릭박스를 클릭시 실행되는 콜백 함수 (Ex. 카테고리 관련 팝업이 나오거나 거래처 관련 팝업이 나온다.)
   */
  callbackFunc: Function;
}
```
