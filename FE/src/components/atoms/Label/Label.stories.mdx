# Label

프로젝트 내에서 사용되어지는 라벨을 표현합니다.

## Props

```ts
interface LabelProps {
  /**
   * Input과 연결되어지는 id 값입니다.
   */
  id: string;
  /**
   * Label의 텍스트 값입니다.
   */
  name: string;
  /**
   * Label의 색상 값입니다.
   */
  color?: string;
}
```
