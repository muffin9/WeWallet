# Input

프로젝트 내에서 사용되어지는 인풋을 표현합니다. (주로 로그인, 회원가입에서 사용)

## Props

```ts
type VariantType = {
  /**
   * 우선 현재는 border 색상과 관련된 버튼의 type. (확장 가능성을 열어두기 위해 type으로 지정.)
   */
  default: string;
  error: string;
  success: string;
};

interface InputProps {
  /**
   * 인풋의 타입을 지정합니다.
   */
  type: string;
  /**
   * 인풋의 이름을 지정합니다.
   */
  name: string;
  /**
   * 인풋의 타입으로 default, error, success가 있습니다.
   */
  variant?: keyof VariantType;
  /**
   * 인풋의 placeholder를 지정합니다.
   */
  placeholder?: string;
  /**
   * 인풋의 value를 바꾸는 onChange 핸들러 함수
   */
  onChange: (e?: React.ChangeEvent<HTMLInputElement>) => void;

  /**
   * 인풋의 value를 지정합니다.
   */
  value?: string | number;
}
```
