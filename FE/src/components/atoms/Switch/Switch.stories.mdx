# Switch

프로젝트 내에서 사용되어지는 스위치 컴포넌트 입니다.

## Props

```ts
interface SwitchProps {
  /**
   * Label과 연결하는 id (라벨 클릭시에도 스위치 동작이 되도록 할 수 있게 하기 위함)
   */
  id?: string;
  /**
   * 스위치의 on/off 상태
   */
  checked?: boolean;
  /**
   * 스위치의 on/off 상태 변경시 호출되는 콜백 함수
   */
  onCheckedChange?: (checked: boolean) => void;
}
```
