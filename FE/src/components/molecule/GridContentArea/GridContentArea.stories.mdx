# GridContentArea

프로젝트 내에서 사용되어지는 그리드 컴포넌트 입니다.(카테고리들을 배치할때 사용)

## Props

```ts
type SizeType = "small" | "medium" | "large";
interface GridContentAreaProps {
  /**
   * 어떤 데이터를 표현할건지에 대한 제목
   */
  title: string;
  /**
   * 그리드를 이루기 위한 key-value 형태로 이뤄져 있는 데이터
   */
  values: { [key: string]: string };
  /**
   * Grid Area Content 영역의 크기를 결정한다.
   */
  size?: SizeType;
  /**
   * Grid Area 영역내에서 특정 아이템 클릭시 해당 아이템으로 value가 변경된다.
   */
  onClick: (key: string) => void;
}
```
