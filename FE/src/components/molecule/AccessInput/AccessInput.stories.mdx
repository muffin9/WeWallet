# AccessInput

프로젝트 내에서 로그인과 회원가입에서 valid를 체킹하여 조합한 인풋 컴포넌트 입니다.

## Props

```ts
/**
 * input의 valid type
 */
type VariantInputType = "default" | "error" | "success";

interface AccessInputProps {
  /**
   * input의 valid type으로 default, error, success가 있습니다.
   */
  variant?: VariantInputType;
  /**
   * Input Component를 받습니다.
   */
  InputComponent: React.ReactNode;
  /**
   * clear image에 클릭리스너에 바인딩될 함수입니다.
   */
  onResetValue: () => void;
}
```
