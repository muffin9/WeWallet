# AccessLabel

프로젝트 내에서 로그인과 회원가입에서 valid를 체킹하여 조합한 인풋 컴포넌트 입니다.

## Props

```ts
/**
 * Label의 valid type
 */
type VariantLabelType = "default" | "error" | "success";

interface AccessLabelProps {
  /**
   * Label의 valid type으로 default, error, success가 있습니다.
   */
  variant?: VariantLabelType;
  /**
   * Label Component를 받습니다.
   */
  LabelComponent: React.ReactNode;
}
```
